HTML5

웹 표준을 준수하고, 구조화 할 수 있으며, 그에 따라 적절한 시맨틱 태그를 사용하여 마크업 할 수 있습니다.

CSS3 / SCSS

구조에 따라 어울리는 레이아웃으로 flex, grid 등을 사용하여 배치하고, 폰트와 색상 스타일을 적용할 수 있습니다. 미디어쿼리로 반응형 웹을 구현할 수 있습니다.

JavaScript

객체, 배열 등의 자료구조를 이해하고 있습니다. 다양한 함수를 활용한 기능구현 하려고 노력하고 있으며, 외부 API 데이터를 능숙하게 활용할 수 있도록 공부하고 있습니다.

Jquery

자바스크립트의 라이브러리인 제이쿼리를 사용해 본 경험은 적습니다만 특유의 매력을 느껴서 다양한 기능들을 시도하고 있습니다.

React

여러 프로젝트에서 '컴포넌트' 기반으로 작업해 보았고, 다른 라이브러리나 프레임워크도 함께 사용해 보았습니다. 아직은 능숙하게 다룰 수 없어서 공부하고 연습하는 중입니다.

TypeScript

타입스크립트로 정적 타입 지정, 코드 오류 감지, 최신 ECMAScript 기능 구현을 해봤습니다. 아직 관련 지식이 부족하여 배우고 공부하는 중입니다.

Firebase

작업한 프로젝트를 지정 도메인으로 배포하고 수정 할 수 있습니다. 다른 기능들은 더 없는지 찾아보고 공부하는 중입니다.

Git / Git-hub

작업물을 VS Code로 git에 업로드하고, 공유할 수 있습니다. git과 git-hub로 협업 할 수 있습니다.

Figma

UX/UI 디자인에 초점을 두며 실시간 협업, 다양한 벡터 그래픽스 편집 및 프로토타입 기능을 활용할 수 있습니다.

Photoshop
Illustrator

포토샵으로 이미지 수정, 색상 보정, 합성 등을 포함한 다양한 이미지 편집을 할 수 있습니다.
일러스트레이터로 벡터 그래픽 디자인, 로고 제작, 아이콘 디자인, 인쇄 및 웹용 그래픽 생성, 텍스트 조작, 패턴 제작 등을 할 수 있습니다.
