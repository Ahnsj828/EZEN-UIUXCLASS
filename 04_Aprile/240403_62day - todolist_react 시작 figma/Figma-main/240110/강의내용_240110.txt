1.어떤 영역안에 입력하고 싶은 텍스트 혹은 아이콘을 삽입한다

2.Auto Layout 설정
1) 첫번째 이유
(*수동으로 모든 컨텐츠의 간격을 다 조정해야하며, 만약 컨텐츠가 Over하게 되었을 때에는 모든 틀 및 간격이 다 틀어진다)
- 방금 전에 입력한 텍스트와 유사한 패턴을 가지고 있는 복수의 컨텐츠를 반복적으로 만들고 싶고 그리고 각 컨텐츠 간 간격을 일정하게 조정
- 어떤 템플릿을 만들고 나서 해당 템플릿에 텍스트가 증가하거나 감소할 때, 자동으로 간격이 조정될 수 있도록 해주기 위한 목적
- ctrl + d  : 복.붙
- 유사한 패턴을 가지고 있는 자식요소의 컨텐츠를 감싸고 있는 부모요소의 Auto Layout 정렬 
(*Vertical Align : 세로 방향 정렬 / Horizontal Align : 가로 방향 정렬)

2) 두번째 이유
- 반복적인 패턴을 갖고 있는 컨텐츠가 없다고해도 단일 컨텐츠의 상.하.좌.우의 패딩 여백을 줘서 배경 컬러를 적용하고 싶어서 (*안쪽 컨텐츠의 width, height 변경)

*어떤 도형 혹은 섹션을 만들고자 할 때, 안쪽에있는 컨텐츠부터 바깥쪽으로 키워나가는 방법 (O)

*바깥에서부터 안쪽으로 만들어나가는 방법
