원래 배열 : ["a", "b", "c", "d", "e"]

수정 배열 : ["b", "a", "c", "d", "e"]

> 배열에 어떤 메서드 함수!!
splice 배열!
splice(0, 1)
0번째 인덱스에 1개를 잘라오겠다
splice(2, 0, a)
2번째 인덱스에 0개를 잘라오겠다 / 아무것도 잘라 오지 않은상태에서 a를넣겠다


*원시타입 변수의 값들은 non-mutation 기능을 가지고 있는 반면
*참조타입 변수의 값들은 mutation 기능을 가지고 있다
 외부로 인해서 값이 변하는거?


배열의 형태가 => 객체의 형태로
