3. 연결리스트의 형태 : stack => 쌓다, 축척하다
 - 설거지 / 엘리베이터 탑승
 - Fisrt In Last Out : 선입후출

 - push( ) : 데이터 삽입 기능
 - pop( ) : 데이터 제거 기능
 - peek( ) : 데이터 참조 기능
 - isEmpty( ) : 데이터가 비어있는지에 대한 여부 체크 기능

데이터 입.출력 통로 1차선인 경우, stack형식의 연결리스트를 사용한다
