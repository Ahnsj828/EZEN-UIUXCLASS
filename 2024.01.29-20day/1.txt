    등등 스타일을 적용하는 역할

3. jbs : 프로그래밍 언어 중 하나다(프로그래밍 언어는 문법이랑 다르다)
  => 여러가지의 언어들이 있다. 
     웹브라우저 화면 ui를 컨트롤 하기 위해서 반드시 초급단계에서 
     공부해야하는 js! 
     더 나아가서는 서버를 구현하기 위한
      java / c / c++ / c# / python / 아두이노 / 바벨 / 웹팩 등등 있다.
  => '공은 이렇게 차는거야' 라고 배우면
  => A의 상황에서는 '공을 살짝 힘을 줘서 차야지'
  => B의 상황에서는 '공을 살짝 힘을 빼고 차야지'
   => 상황에 따라 다르게 나온다 -> 상황에따라 다른 판단이 나올 수 있다

코드를 작성하기에 앞서서 내가 현재 구현하고자 하는 기능이 무엇인지부터
정의하고 가는게 좋다!!!
 1) 컴퓨터에게 우측상단 토글 버튼이 존재한다. 라는것을 알려줘야한다
     - 자바스크립트에서 => 변수(*바구니)를 선언한다 => 토글버튼을 답아줘야한다
 2) 컴퓨터에게 gnb 네비게이션이라는 존재 역시 알려줘야 함
    - 자바스트립트에서 => 변수를 선언 => gnb 네비게이션을 담아준다
 3) 우측상단 토글 버튼을 클릭하지 않은 상태일 때에는 gnb 네비게이션이 
     나타나면 안된다.
    - css => 특정요소의 존재 자체를 없앨 수 있는 스타일 속성
    (*display : none)
 4) 우측상단 토글 버튼을 클릭했을 때, gnb 네비게이션이 나타난다.
    - 자바스크립트에서는 이벤트가 존재한다
      => 이벤트 - User가 어떤 액션 행동을 취했을 때 어떠한 행위를 만들어라
          (이벤트로 인해서 리액션을 만들어라!!)
          addEventListner라는 함수(*어떤 기능을 정의해 놓은 역할)를 사용할 수 있다.
