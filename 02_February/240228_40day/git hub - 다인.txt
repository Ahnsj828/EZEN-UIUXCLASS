1. 1:1 방식으로 git-hub 사용방법
    - 로컬컴퓨터 & 레포지토리 -연결되어있어야한다
    -  대표/사수/선임 <-> 신입사원/사수/직원
    - 개인적으로 뜻이 맞는 사람들끼리 스터디 & 프로젝트
    - git pull 과 git clone 차이
      커밋되어져있는 파일을 가져오는건 똑같다
      git pull : 작업 중간 버전 모든 데이터를 다운받겠다
                  몇번 올리고 수정되어있는걸 받고 작업하겠다
                  몇번 했다할때 사용

      git clone :  누가 만든거를 받겠다
                    이제 막 시작하는단계에 남이 만든거를 토대로 만들겠다
                     최초 작업할때 사용
