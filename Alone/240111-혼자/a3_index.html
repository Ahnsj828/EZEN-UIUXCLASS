Do it! HTML+CSS+자바스크립트 웹 표준의 정석

구매하기

저자 정보 (2021) 저자는 2010년 《Do it! HTML5 + CSS3》 출간 이후 꾸준히 HTML5 강의를 해왔다. 자타공인 HTML 권위자인 저자는 웹 초창기부터 ASP와 JAVA 등 전문 도서를 번역하면서 웹 기술에 관심을 가지기 시작했다. 이후 HTML과 DHTML(Dynamic HTML), CSS, JavaScript 등 웹 관련 기술을 좀 더 쉽게 익힐 수 있는 도서 60여 종을 집필했다. 홍익대학교 컴퓨터공학과를 졸업하고 동 대학원에서 인공지능을 공부한 뒤 현재 Visual Studio and Development Technologies 분야에서 마이크로소프트 공인 최고 기술자(Microsoft Most Valuable Professional)로 활동하고 있다. 이 책은 《Do it! HTML5 + CSS3 웹 표준의 정석》과 《Do it! 자바스크립트 - 기본 편》의 합본호 출간을 희망하는 많은 독자의 요구를 반영하여 탄생했다.

대표 저서 • 《Do it! 지옥에서 온 문서 관리자 깃 & 깃허브 입문》 • 《Do it! 웹 프로그래밍을 위한 자바스크립트 기본 편》 • 《윈도우 10 무작정 따라하기 – 최신 개정판》 • 《Do it! 프런트엔드 웹 디자인 입문 - 전면 개정판》 • 《Do it! HTML5 + CSS3 웹 표준의 정석 – 전면 개정판》 • 《Do it! HTML5 + CSS3》(단기 완성) - 전자책 등 총 60여 종
